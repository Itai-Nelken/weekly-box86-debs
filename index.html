To add this repo to your system, run:

wget -qO- https://itai-nelken.github.io/weekly-box86-debs/add-repo.sh | sudo bash

Manual installation
sudo wget https://itai-nelken.github.io/weekly-box86-debs/debian/box86.list -O /etc/apt/sources.list.d/box86.list

wget -qO- https://itai-nelken.github.io/weekly-box86-debs/debian/KEY.gpg | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/weekly_box86_debs_keyring.gpg

Remember to run sudo apt update after doing that to refresh the reositories.

Installing box86:

After adding the repository to your system, you can install box86 by running sudo apt install box86.

Installing box64

Box64 is not available in this repo, however you can use this repo by ryanfortner on github here.

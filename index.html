weekly-box86-deb's-apt-repo

weekly box86 debs

To add this repo to your system, run:

wget -qO- https://itai-nelken.github.io/weekly-box86-debs/add-repo.sh | sudo bash
Manual installation
deb https://itai-nelken.github.io/weekly-box86-debs/debian/ /
Remember to run
sudo apt update
after doing that.
